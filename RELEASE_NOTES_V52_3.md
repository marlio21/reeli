# v52.3 – Simple Card Builder

This release resets the mobile Studio editor away from menu experiments and toward a simple card-builder flow inspired by modern digital business card apps.

## Changed

- Replaced the mobile-only visible menu controls with a block-based Card Builder.
- The mobile Studio now shows clear editing sections below the compact 9:16 preview:
  - Video / Scene
  - Werbetext
  - Aktionen / Buttons
  - Design
  - Teilen / QR
- Buttons are now shown as large editable action cards.
- The active button has a large live preview plus direct fields for:
  - visible button text
  - second line
  - action type
  - action target
  - button color and text color
  - design transfer to all buttons
- Text editing is simplified to title, subtitle, description and a few presets.
- Scene editing includes video URL, duration, remove video and simulation.
- Design is simplified to core presets.
- Public Link and desktop renderer were intentionally left unchanged.

## Intent

The mobile Studio should feel like editing a digital card with actions, not like navigating abstract menus.

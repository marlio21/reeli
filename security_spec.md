# Security / Storage Notes — v19

Diese Version fügt zwei lokale SVG-Assets im `public`-Ordner hinzu. Es werden dafür keine neuen Storage-Regeln benötigt.

Die echten Button-Ziele bleiben leer, bis der Nutzer Telefonnummer, Webseite, Mail, WhatsApp, Unternehmensdaten oder Datei/PDF ergänzt.

Für spätere Datei-/Bild-Uploads gelten weiterhin die bestehenden Firebase Storage-Regeln.

# v52.2.2 – Compact 9:16 Preview & Big Button Editor

Mobile Studio polish after v52.2.1.

## Goals
- Keep the Studio preview as a real 9:16 card, compact and fully visible.
- Make the Button Studio easier to understand by showing one large active button as the current work object.
- Keep the simple dashboard direction: no Orbit, no transparent overlay chaos.
- Keep Public Link / Desktop unchanged.

## Changes
- Tightened mobile Studio preview height.
- Preserved 9:16 aspect ratio via aspect-ratio rules.
- Reduced mobile subnavigation chip size so more tools fit.
- Button submenu now exposes: Button wählen, Text, Aktion, Look, Icon, Größe, Übertragen.
- Added mobile-only large active button workspace above button controls.
- Added a horizontal button strip to switch the active button quickly.
- Increased non-compact button preview tile size for a more tactile editor feel.

# ureel.me Studio v17

Änderungen:
- Werbetexter-Timer mit Block-Modus: ganze Werbevorlage gemeinsam timen oder einzelne Textfelder separat timen.
- Titel, Slogan und Beschreibung bleiben einzeln deaktivierbar; leere Felder werden nicht angezeigt.
- Schriftgröße, Schriftart und Schriftfarbe sind direkt im Werbetext-Menü sichtbar.
- Animationsdauer im Bereich Rahmen & Stil bis 3 Sekunden einstellbar.
- Timing synchronisiert die Vorschau über `profileTextReveals` inklusive Fade-Dauer.

Build geprüft: `npm run build` erfolgreich.

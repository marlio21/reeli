# v52.4.1 – Mobile Button Editor Polish

Diese Version fokussiert bewusst nur den mobilen Button-Editor innerhalb des Tap-to-Edit Mobile Studios.

## Änderungen

- Horizontale Button-Auswahlreihe unter „Tippe einen Button und bearbeite ihn“ entfernt.
- Buttonbearbeitung wird stärker auf den direkt ausgewählten Button in der 9:16-Karte ausgerichtet.
- Button-Panel reduziert auf klare Werkzeuge: Text, Aktion, Look, Übertragen.
- Look-Bereich erweitert:
  - breitere Farbauswahl für Buttonfarbe,
  - Textfarbe,
  - Form,
  - Größe,
  - Icon,
  - Rahmen.
- Button-Größe skaliert nun nicht nur die Fläche, sondern auch Schrift, Icon, Innenabstand, Rahmen und Radius mit.
- Schwarzes Vollbild-Speicher-Overlay durch einen kleinen „Gespeichert“-Hinweis ersetzt.
- Änderungen bleiben live auf Arbeitsbutton und 9:16-Vorschau ausgerichtet.

## Unverändert

- Public Link bleibt unverändert.
- Desktop bleibt unverändert.
- Tap-to-Edit-Grundlogik aus v52.4 bleibt erhalten.

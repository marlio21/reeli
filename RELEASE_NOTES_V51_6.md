# v51.6 – Mobile Studio Structure Redesign

Ziel: Das mobile Studio wird klarer, heller und besser bedienbar. Die öffentliche Live-Karte bleibt unverändert.

## Änderungen

- Hauptnavigation mobil als 4 sichtbare Module nebeneinander: Szene, Timeline, Buttons, Design.
- Vorschau im Studio kompakter, damit die Bearbeitung schneller erreichbar ist.
- Untermenü als helle horizontale Chip-Leiste statt dunkler verschachtelter Liste.
- Detailbereich auf Mobile hell/creme mit dunkler Schrift.
- Formularfelder größer und besser lesbar.
- Kartenverwaltung im mobilen Bearbeitungsfluss weiter ausgeblendet; erreichbar über Kontomenü.

# ureel.me – v52.5.0 Card & User Management Foundation

Diese kompakte Upload-Version startet die Kartenverwaltung und Nutzerverwaltung, ohne den zuletzt stabilisierten mobilen Karteneditor, den Public Link oder die Desktop-Ansicht grundlegend umzubauen.

## Neu in v52.5.0

- Dashboard-Tab **Meine ureels** bleibt der zentrale Einstieg für Kartenverwaltung.
- Kartenaktionen sichtbarer: neue Karte, bearbeiten/öffnen, Live-Link öffnen, duplizieren, löschen.
- Neuer Dashboard-Tab **Konto** als Grundlage der Nutzerverwaltung.
- Konto zeigt Account, Plan, Kartenlimit, Name und Firma/Organisation.
- Name/Firma können als Basisdaten gespeichert werden.
- Alte `RELEASE_NOTES_*.md` bleiben aus dem Upload entfernt; Verlauf läuft über `CHANGELOG.md`.

## Geschützt

- Mobiler Karteneditor bleibt weitgehend unangetastet.
- Public Link bleibt unverändert.
- Desktop-Ansicht bleibt unverändert.
- Keine echten Secrets in der ZIP.

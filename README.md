# ureel v35 – Desktop-Webseite und Replay

Diese Version erweitert den Designbereich zur Desktop-Miniwebseite und verbessert die Bedienung der Kartenverwaltung sowie den Neustart eines abgespielten Spots.

## Änderungen

- Desktop-Designvorschau als 3-Spalten-Aufbau: Smartphone / Werbetext / Buttonbereich.
- Nutzerbuttons können dauerhaft neben dem Smartphone angezeigt werden.
- Buttonbereich der Desktop-Webseite bekommt eigenen Hintergrund: ohne / Verlauf / Bild-Upload.
- Werbetexter kann direkt aus dem Designbereich geöffnet werden.
- Kartenverwaltung kann per X, Hintergrund-Klick und zusätzlichem Schließen-Button geschlossen werden.
- Nach Ablauf/Endkarte gibt es in der Karte einen Button „Spot neu starten“, der Timeline und Video neu startet.

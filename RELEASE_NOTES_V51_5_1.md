# v51.5.1 – Mobile Studio Hard Layout Fix

Ziel: Die öffentliche Karte bleibt unverändert. Das mobile Studio wird deutlicher verbessert, weil v51.5 optisch zu wenig Wirkung hatte.

## Änderungen

1. Mobile Studio-Untermenüs sind jetzt horizontale Chip-Leisten statt langer vertikaler Kartenlisten.
2. Der große Bereich „Aktive ureel-Seite / Neue ureel / Karten verwalten“ wird im mobilen Submenü ausgeblendet, damit die eigentliche Bearbeitung schneller erreichbar ist.
3. Detail-Header im Studio ist wieder gut lesbar. Titel und Erklärung bleiben hell auf dunklem Hintergrund.
4. Vorschau bekommt etwas mehr echten Karteninhalt und weniger dekorativen Handyrahmen.
5. Formular-Karten bleiben hell/cremefarben mit dunkler Schrift.
6. Public Live-Link / öffentliche Karte wurde nicht verändert.

## Testpunkte

- Am Smartphone Szene / Timeline / Buttons / Design durchklicken.
- Prüfen, ob die Untermenüs nun kompakter und schneller bedienbar sind.
- Prüfen, ob die Detail-Überschrift wieder lesbar ist.
- Öffentlichen Link kopieren und sicherstellen, dass die schöne Live-Karte unverändert bleibt.
